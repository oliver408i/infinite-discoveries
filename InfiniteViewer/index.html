Infinite Viewer

A simple three.js based star system viewer for KSP (Kopernicus)

Warning!
This is a work in progress, it is not 100% accurate to what you will see in KSP!
Use this only as a quick visual check only, or to preview a pack before you load it up. Nothing will beat actual KSP!
Might not work with all planet packs!

Features implemented

  • Planetary bodies
  • Texture, terrain, and heightmap for planets
  • Normal maps (WIP, still very buggy)
  • Orbits (and their lines)
  • Orbit attributes
  • Stars and approx light
  • Star light colors (on planets only)

Features not implemented

  • Atmospheres
  • Planetary rings (To be added)

How to use
Use mouse to pan around. Click a planet's name to focus on it.
Press TAB to go to next planet (shift+TAB not implemented yet).
The camera placement is a bit weird, for stars and big planets, you may need to zoom out to see it.
If you don't see anything on load, press Tab to focus on the next body

Select a zip file to load. The .cfg and textures can be anywhere in the zip file
.dds files CANNOT have duplicate names! All filenames must be unique for dds files!
